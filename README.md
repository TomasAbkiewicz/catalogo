# 🎬 CineScope - Catálogo de Películas

Catálogo de películas desarrollado con Astro como actividad diagnóstico de Frontend.

## Herramientas usadas
- [Astro](https://astro.build/)
- HTML semántico
- CSS Grid
- JavaScript vanilla

## Funcionalidades
- Listado de películas
- Búsqueda en tiempo real
- Filtros por género
- Dark / Light mode
- Diseño responsive

## Cómo ejecutarlo

1. Clonar el repositorio:
```bash
   git clone https://github.com/TU_USUARIO/catalogo.git
   cd catalogo
```
2. Instalar dependencias:
```bash
   npm install
```
3. Iniciar el servidor:
```bash
   npm run dev
```
4. Abrir en el navegador: `http://localhost:4321`
